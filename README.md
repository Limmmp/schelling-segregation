# Модель сегрегации Шеллинга

Агентное моделирование на сетке: два типа агентов, параметр «терпимости»,
анимация перемещений, измерение индекса сегрегации по поколениям и
построение зависимости сегрегации от терпимости.

Программа кроссплатформенная: работает на Windows и Linux (а также macOS)
без изменения кода. Требуется Python 3.9+ и библиотеки numpy, matplotlib.

Готовые исполняемые файлы (не требуют установленного Python) лежат в
каталоге `dist`: `schelling-win.exe` для Windows и `schelling-linux` для
Linux. Параметры командной строки у них те же, что и у `schelling.py`,
например: `schelling-win.exe experiment --runs 5`.

## Установка

Windows (PowerShell / cmd):

```
python -m pip install -r requirements.txt
```

Linux:

```
python3 -m pip install -r requirements.txt
```

Примечание для Linux: для показа окна анимации нужен пакет tkinter
(в Debian/Ubuntu: `sudo apt install python3-tk`). Режимы `--gif` и
`experiment` работают и без него.

## Запуск

Анимация модели (окно с сеткой и графиком индекса сегрегации):

```
python schelling.py animate --size 50 --density 0.9 --tolerance 0.4
```

Сохранение анимации в GIF (без показа окна):

```
python schelling.py animate --tolerance 0.4 --steps 60 --gif demo.gif
```

Эксперимент — зависимость итоговой сегрегации от терпимости
(строит график PNG и таблицу CSV):

```
python schelling.py experiment --runs 5 --out segregation_vs_tolerance.png
```

На Linux вместо `python` используйте `python3`.

## Параметры модели

| Параметр      | Смысл                                             | По умолчанию |
|---------------|---------------------------------------------------|--------------|
| `--size`      | размер сетки NxN                                  | 50           |
| `--density`   | доля занятых клеток                               | 0.9          |
| `--ratio`     | доля агентов первого типа                         | 0.5          |
| `--tolerance` | терпимость T — допустимая доля «чужих» соседей    | 0.4          |
| `--steps`     | максимальное число поколений                      | 200          |
| `--seed`      | зерно случайных чисел (для воспроизводимости)     | —            |

Агент недоволен, если доля соседей другого типа среди его соседей
(окрестность Мура, 8 клеток) превышает T. Все недовольные агенты
одновременно переезжают в случайные свободные клетки — это одно
«поколение» модели.

Индекс сегрегации — средняя по всем агентам доля соседей своего типа:
около 0.5 — полное перемешивание (при равных долях типов), 1.0 — полная
сегрегация.
